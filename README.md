# production-practice

Учебный репозиторий по производственной практике.

В репозитории размещена программа `shift_calculation.py`, разработанная для моделирования учета выполнения сменного задания на предприятии ООО «Татнефть-Самара».

Функции программы:
- ввод наименования продукта;
- ввод планового и фактического объема выпуска;
- расчет процента выполнения плана;
- вывод результата на экран и запись в текстовый файл.
